What to Check When Choosing LMS

A beginner's guide to choosing a Learning Management System.

Need to pick a Learning Management System (LMS) but are not sure what to look for? We’ve got you covered. This handy checklist has all the tips and tricks you need to choose the LMS that is right for you. Use it when evaluating an LMS to make sure that it has all the features you need.

  • A cloud LMS can be deployed much quicker than an on-premises counterpart, requires no maintenance, and does not create more work for your IT staff. On top of that, a cloud LMS can be accessed from anywhere in the world and is always up-to-date.

  • Today, some LMSes come bundled with advanced course building tools that pack all essential features, and often much more besides. Unless you have very specific needs that can only be fulfilled by using a highly specialized course building suite, you can do just fine working in an LMS featuring a built-in course builder.

  • Advanced course builders allow you to create conditional, nonlinear connections between individual course objects, which makes it possible to create courses that adapt to every learner’s individual degree of understanding.

  • The ability to add stand-alone questions and assessments to the course. It allows you, for example, to create a pre-test to assess the learner’s knowledge before starting a course, and then add a final test to check how well he understood the course content.

  • Reusing existing elements can save you a lot of time and effort. When building online courses on a regular basis, the ability to import objects from earlier courses that have already proved their worth is a life saver.

  • Surveys help to collect feedback about courses. You will be able to track the responses of users either anonymously or not, see if they skipped some questions or not. Finally, see responses of a user to all questions or responses of all users to a specific question.

  • Homework helps you to introduce hands-on practice tasks into your courses. Unlike tests, homework assignments are uploaded as text or as a file that is then evaluated by the teacher, and students proceed along the “pass” or “fail” paths depending on the results of the evaluation.

  • The ability to assign roles and privileges to users and groups. Every role comes with a list of predefined privileges. Privileges determine what functionality is available to users sharing a particular role. For example: Administrator, Manager and Learner. Does the LMS allow to create custom roles?

  • Check if the LMS allows to create not only groups, but also subgroups that can be members of a different group. This enables you to recreate within the LMS the organizational structure of the company’s business units.

  • Filters and sorting mechanisms that help you save time when searching for users, groups and courses by given parameters. Sometimes you may need to narrow the current scope down to a certain set of users and groups that share a specific criterion.

  • The ability to add custom fields to users and groups without the need for assistance from the vendor. Use these custom fields to store, track, and search through custom attributes, such as Birth Date, social media profile links, etc.

  • The ability to issue certificates giving proof of achieving the required level of competence in a certain area or sufficient understanding of a particular topic or subject. Managers can plan the process of employee training and schedule refresher courses around the certificates’ expiration dates. Awarding certificates motivates the staff and helps to more closely align the learning process with the enhancement process within the company.

  • Help Desk support staff that speaks English as their primary language and replies within 48 hours.

  • The option to get dedicated support guaranteeing direct access to a technical team who can investigate any problem and propose multiple solutions if needed.

  • Is there an option to track the students’ learning progress and receive detailed statistics about the results of a specific user? Can you add tags to questions to track the results in specific areas of expertise? With group statistics you can identify the effectiveness and success rates of learning programs and make adjustments as needed.

  • It is important to keep on top of things when managing a large number of learners. Has a particular learner completed the course you assigned to them? Did the new hire create an account? With reports, all relevant data is at your fingertips.

  • Some LMSes allow you to create storyboards. Storyboarding is a method for creating the framework of a course by arranging blocks representing the main parts of the course. It is important that the storyboard clearly shows how all pages fit together and the order in which the learner taking the course proceeds from one page to another via linear and nonlinear transitions.

  • The ability to communicate with other users and trainers via chats to ask questions, get help and provide feedback.

  • Allows users to initiate and participate in discussions in different contexts, such as in-group discussions, chat rooms dedicated to specific courses, user-to-user messaging or custom chat rooms.

  • The ability to broadcast messages either to all users or to selected users and groups only. In addition to delivering messages from the administration, this feature can be used, for example, to notify users by Email about incoming messages.

  • Manage interface branding themes and specify user settings. Add the company's logo and change elements' colors.

  • Usually, LMSes come with support for several interface languages that you or your users can choose from. Check if this setting is bound to a user account, so that every user is free to choose the preferred language of the web interface, or if it is bound to the instance and only the administrator can set the system language.

  • Cloud LMS can be deployed in a fraction of the time, requires no maintenance, and leaves your IT people free to carry out their regular duties. On top of that, a cloud LMS can be accessed from anywhere in the world and is always up-to-date.

  • Responsive Learning Management System provides an optimal viewing and interaction experience. It allows easy reading and navigation with a minimum of resizing, panning, and scrolling, across a wide range of devices, like desktops, tablets and smartphones.

  • Allows to save created courses in SCORM format. SCORM is a collection of specifications that enable interoperability, accessibility and reusability of web-based learning content. SCORM content can be delivered to learners via any SCORM-compliant Learning Management System.

  • This feature allows you to import content from other popular course-building solutions instead of creating it from scratch. Make a note of the formats the LMS supports as sources for import. Does it support SCORM? PDF? PPTX?

  • API stands for Application Programming Interface. It is a set of pre-packaged instructions meant to make the programmer’s life easier when they need to integrate one product with another. It enables integration and interaction between the LMS and other software solutions used in the company.
